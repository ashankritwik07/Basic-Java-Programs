Question:-
1. You are required to print a 'z' of size 5 using '*'.
Sample Input:

There is no input

Sample Output:

*****
 *
 * 
* 
*****

Solution:-

import java.util.*;

public class Main {

  public static void main(String[] args) {
    System.out.println("*****\n   *\n  *\n *\n*****");
   
  }
}

EXPLANATION:- here we can use (\n) to change the line instesd of using the general meathod (println).
